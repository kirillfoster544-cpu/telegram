# Telegram Bot — без кнопки подтверждения

⚠️ При /start бот сразу предупреждает:
- Получатель видит сообщение как анонимное
- Админ видит: кто написал, кому написал и текст

## Railway Variables
- BOT_TOKEN = (твой токен — лучше отозвать старый и поставить новый)
- ADMIN_ID = 7489815425

## Команды
- /start
- /admin (только для ADMIN_ID)

## Как писать
1) Человек берёт «Моя ссылка»
2) Друг открывает ссылку и пишет сообщение
3) Бот отправляет получателю анонимно + копию админу
